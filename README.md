# DESCRIPTION OF TASK PERFORMED :

As part of my Full Stack Web Development Internship at Codtech IT Solutions, I developed a Chrome Extension for Time Tracking and Productivity Analytics to monitor user activity and improve productivity.

The objective of this project was to track the time spent on different websites and provide insights into user browsing behavior. The extension captures active tab data, records usage time, and categorizes productivity levels based on website activity.

On the frontend, I designed the user interface using HTML, CSS, and JavaScript to display tracked data in a clear and user-friendly format. The extension includes a popup interface that allows users to view their activity and productivity statistics.

On the backend, I implemented logic using Node.js and JavaScript to handle data processing and storage. I also structured the extension using key Chrome components such as manifest.json, background scripts, and content scripts to ensure proper functionality.

Key features implemented include:

Tracking time spent on different websites
Monitoring active tabs in real-time
Displaying productivity analytics using charts
Storing and retrieving user activity data

Additionally, I worked with files such as background.js, content.js, popup.js, and chart.js to manage extension behavior and data visualization.

This project helped me understand browser extension architecture, event handling, and real-time data tracking. It also enhanced my skills in JavaScript, full stack integration, and performance optimization.

# OUTPUT OF THE TASK

The final output is a fully functional Chrome Extension that tracks user activity and analyzes productivity based on time spent on websites. It provides real-time insights through a popup interface and visual charts, demonstrating effective browser integration and data handling.
